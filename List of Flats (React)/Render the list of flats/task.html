Description

In this stage, you are tasked with showing the list of flats with their details, including name, location, price, and availability status. The app should include header and main tags. The header should have a <h1> title and the main should have a <h2> title and an <ul> tag for the list of flats.

Use the below array of objects for the flats' details:

 [
        {
            id: 1,
            name: 'Cozy Apartment',
            location: 'Downtown',
            price: '$1500/month',
            available: true,
            image: 'https://example.com/cozy-apartment.jpg',
        },
        {
            id: 2,
            name: 'Modern Loft',
            location: 'Midtown',
            price: '$1800/month',
            available: false,
            image: 'https://example.com/modern-loft.jpg',
        },
        {
            id: 3,
            name: 'Charming Studio',
            location: 'Uptown',
            price: '$1200/month',
            available: true,
            image: 'https://example.com/charming-studio.jpg',
        },
  ]

Pay attention to the proper use of HTML semantics and the objectives below.

Objectives

  1. Component Structure:

    • Develop a functional React component named App as the main application container.

    • Import necessary React libraries and modules.

    • The main application container div should have App as a class.

  2. Header Section:

    • Implement a header section using the <header> HTML tag as the container.

    • Include an <h1> tag for the title and name your application such as "SuperFlats".

  3. Main Section:

    • Create a main section within the <main> HTML tag to house the list of flats.

    • Include an <h2> tag for the flats list.

    • The text should say "Flat List".

  4. Flats List:

    • Display a list of flats using the <ul> (unordered list) HTML element.

    • Iterate over the flats array, rendering a list item (<li>) for each flat.

  5. Flat Information:

    • For each flat item, display information such as the flat's name, location, price, availability status, and an image.

    • The flat name should be a <h3> tag while the location, price, and availability should be a <p> tag.

  6. Availability Status:

    • Dynamically display the availability status based on the available property of each flat. If available is true, show "Available"; otherwise, show "Not Available."

  7. Image Display:

    • Display the image of each flat using the <img> HTML element.

    • Add the alt attribute to the image for accessibility with the value as the flat name.

  8. Key Attribute:

    • Although tests don't check, it's best practice to assign a unique key attribute to each list item using the id property of the flat object for efficient React virtual DOM updates.
