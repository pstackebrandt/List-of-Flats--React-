Description

In this stage, the focus is on improving the application's functionality by successfully adding a new flat to the existing list after submitting the form. Follow these requirements to achieve this:

Objectives

  1. Input Change Handling:

    • Implement a function (handleInputChange) to update the state of the form inputs as the user enters information.

  2. Checkbox Handling:

    • Ensure the checkbox for availability updates the available state in the form.

  3. Add Flat Button:

    • Create a button within the form labeled "Add Flat" that, when clicked, adds the new flat to the flats array.
    • Implement a function (handleAddFlat) to handle the addition of the new flat and reset the form state.

  4. Form Reset:

    • After successfully adding a new flat, reset the form to clear the input fields.

  5. Display New Flat:

    • Verify that the newly added flat appears in the list along with the existing flats.
