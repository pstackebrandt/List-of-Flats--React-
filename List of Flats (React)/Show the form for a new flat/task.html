Description

In this stage of the project, your objective is to enhance the application by allowing users to add more flats dynamically. Implement a button that, when clicked, reveals a form to input details for a new flat. Follow these objectives to achieve this functionality:

Objectives

  1. Show Form Button:

    • Implement a button with the ID addFlatButton labeled "Add Flat" that, when clicked, toggles the visibility of the form.
    • Conditionally render the form or the button.

  2. Form Display:

    • Develop a form that includes input fields for the name, location, price, image URL, and availability of the new flat.
    • Use appropriate HTML input elements (text, checkbox) and labels for each field.
    • Use the name attribute for inputs, they should be: name, location, price, image, and available.

  3. Form Visibility Toggle:

    • Toggle the visibility of the form when the "Add Flat" button is clicked. The form should be hidden by default and displayed when the button is clicked.
    • The form and button should be in the main tag and should appear before the "Flat List".

  4. Display Existing Flats:

    • Continue displaying the existing flats in the list below the form.
